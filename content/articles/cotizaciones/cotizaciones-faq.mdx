**¿Cuántas cotizaciones puedo enviar para una misma solicitud?**
Solo una activa a la vez. Si quieres ajustar tu precio o detalles, puedes retirar la cotización y enviar una nueva mientras el cliente no haya aceptado otra.

**¿Puedo retirar una cotización que ya envié?**
Sí, siempre que el cliente no la haya aceptado. Ve a la solicitud → tu cotización → "Retirar cotización".

**¿El cliente ve si modifiqué o retiré mi cotización?**
El cliente ve la versión actual de tu cotización y recibe una notificación si la actualizas.

**¿Cuántos maestros cotizan para la misma solicitud?**
Depende del trabajo. Generalmente varios maestros cotizan para la misma solicitud — por eso la calidad del perfil y la nota al cliente marcan la diferencia.

**¿Puedo ver qué cotizaron los otros maestros?**
No. Las cotizaciones de otros maestros son confidenciales para ti. El cliente puede ver todas las cotizaciones que recibió.

**¿Qué pasa si el cliente no responde mi cotización?**
La cotización queda activa hasta que el cliente acepte alguna, cancele la solicitud, o el proyecto expire por inactividad.

**¿Puedo cotizar en cuotas?**
Manito maneja tramos de pago (etapas del trabajo), pero no cuotas de financiamiento. El monto total debe pagarse en Manito — lo que varía es cuándo se libera cada parte.

**¿El precio de la cotización incluye IVA?**
Puedes configurarlo desde el compositor. El sistema puede mostrar precio con o sin IVA — el cliente verá el desglose completo.

**¿Se pueden agregar cargos adicionales después de que el cliente aceptó?**
No directamente. Si surge un imprevisto, debes crear una **orden de cambio** para que el cliente apruebe el cargo adicional.

**¿Cuánto tiempo tengo para enviar una cotización desde que veo la solicitud?**
No hay plazo fijo, pero las solicitudes tienen vida útil. Cotiza lo antes posible para no perder la oportunidad — los clientes suelen elegir rápido.
